线性代数深度解析：二次型与矩阵变换

一次关于伴随矩阵、正交变换和谱定理的精彩讨论记录

一、 原始问题与标准解答

题目： 已知三元二次型 xᵀAx 经正交变换化为 -y₁² - 2y₂² - y₃²，其中 Aᵀ = A。则二次型 xᵀA*x 的正惯性指数为多少？（A* 为 A 的伴随矩阵）

标准解题步骤：

  1. 求 A 的特征值 (λ)： 标准型中的系数即为特征值。所以 A 的特征值为 -1, -2, -1。
  2. 求 A 的行列式 (|A|)： 行列式等于特征值的积。|A| = (-1) * (-2) * (-1) = -2。
  3. 求 A* 的特征值 (λ*)： 根据公式 λ* = |A| / λ（假设 A 可逆）。
    • λ₁* = -2 / -1 = 2
    • λ₂* = -2 / -2 = 1
    • λ₃* = -2 / -1 = 2
  4. 求正惯性指数： A* 的特征值为 2, 1, 2，都是正数。因此正惯性指数为 3。

二、 深度探索 (1)：不记公式，如何推导？

如果忘记了 λ* = |A| / λ 这个公式怎么办？我们可以从最基本的定义出发。

核心工具箱：

  1. 特征值的定义： Av = λv (v 是特征向量)
  2. 伴随矩阵的定义： A*A = AA* = |A|I (I 是单位矩阵)

推导过程：

目标是找到 A* 的特征值。我们来看 A* 作用在 A 的特征向量 v 上会怎样。

1. 从定义出发： Av = λv

2. 等式两边左乘 A*： A*(Av) = A*(λv)

3. 化简左边（使用 A*A = |A|I）：
   (A*A)v = (|A|I)v = |A|v

4. 化简右边（λ 是常数）：
   A*(λv) = λ(A*v)

5. 结合两边： |A|v = λ(A*v)

6. 整理成特征值形式 (A*v = λ*v)：
   A*v = (|A| / λ) v

结论： A 的特征向量 v 也是 A* 的特征向量，且对应的特征值 λ* = |A| / λ。我们成功地自己证明了这个公式！

三、 深度探索 (2)：用户的巧妙思路 (使用逆矩阵 A⁻¹)

用户提问： “我能不能得到 A⁻¹ 的正交化矩阵，然后再间接得到 A* 的正交化矩阵？”

解析： 这个思路在理论上是完全正确的，并且展现了对线性代数结构的深刻理解。

理论推导：

  1. A 的对角化： 实对称矩阵 A 可被正交矩阵 P 对角化：A = PDPᵀ (D 是特征值对角阵)。
  2. A⁻¹ 的对角化： A⁻¹ = (PDPᵀ)⁻¹ = (Pᵀ)⁻¹ D⁻¹ P⁻¹ = P D⁻¹ Pᵀ。
    (关键发现：A 和 A⁻¹ 被同一个正交矩阵 P 对角化)
  3. 连接 A* 和 A⁻¹： 根据定义 A⁻¹ = (1/|A|) A*，即 A* = |A| A⁻¹。
  4. A* 的对角化： A* = |A| (P D⁻¹ Pᵀ) = P (|A|D⁻¹) Pᵀ。
    (结论：A* 也被 P 对角化，其特征值对角阵为 |A|D⁻¹)。

为何实战中不用？ 尽管漂亮，但此方法需要知道矩阵 P，而题目未给出。直接用特征值关系更快捷。

四、 核心概念辨析：QᵀAQ 与 Q⁻¹AQ

用户提问： “有的时候可以看到 QᵀAQ = Q⁻¹AQ，请问这种条件一般怎么产生？”

答案： 当且仅当 Q 是正交矩阵 (Orthogonal Matrix) 时，Q⁻¹ = Qᵀ。

两种变换的本质区别：

  • 相似变换 (B = Q⁻¹AQ)： 用于研究线性变换在不同基下的表示。保持特征值不变。
  • 合同变换 (B = QᵀAQ)： 用于研究二次型。保持正负惯性指数（正负特征值个数）不变。

当处理实对称矩阵 A 时，我们可以找到一个正交矩阵 Q 来对角化它。此时，相似变换和合同变换合二为一，都得到了相同的对角矩阵（特征值矩阵）。这大大简化了问题。

五、 终极提问：为什么实对称矩阵如此特殊？

用户提问： “为啥 A 是实对称矩阵就可以用正交矩阵对角化它？如果不是，就没有正交矩阵了吗？我可以通过变换将其变为实对称矩阵吗？”

谱定理 (Spectral Theorem) 的魔力：

这是线性代数的核心定理之一。实对称矩阵之所以特殊，是因为它拥有两个关键性质：

  1. 性质一： 实对称矩阵的特征值必定是实数。
  2. 性质二（关键）： 实对称矩阵对应于不同特征值的特征向量必定相互正交。
    （正是这些正交的特征向量构成了我们的正交矩阵 P。）

如果不是实对称矩阵？

不能用正交矩阵对角化。 它的特征向量通常不正交。

能“变”成对称矩阵吗？

不能。 矩阵的对称性是其内在属性，不随坐标变换（换基）而改变。

但我们可以“提取”： 任何方阵 A 都可以唯一分解为 A = (对称部分) + (反对称部分)。对于二次型 xᵀAx 来说，反对称部分没有贡献，所以研究 xᵀAx 等价于研究 xᵀ(对称部分)x。

总结： 你的所有疑问都指向了线性代数中最优美的部分——对称性带来的秩序。掌握了这些，再复杂的变换问题也能迎刃而解。
